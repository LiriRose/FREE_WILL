חידה מקצועית – מה זה Game Object?
התשובה: ב־Unity, Game Object הוא האובייקט הבסיסי שממנו בנוי המשחק. כל דבר בסצנה (דמות, מצלמה, קובייה, תאורה וכו') הוא Game Object, ועליו מוסיפים Components שמגדירים את ההתנהגות והתכונות שלו.
